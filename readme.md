# Store moke app

This is a simple Python program that registers products and calculates the total sales of the day. The project is organized into three files:

**main.py**
Runs the program, shows the date and time, and calls the other modules.

**products.py**
Handles product registration. It asks the user for the product name, price, and quantity, and stores them in a list.

**sales.py**
Calculates the sales summary by multiplying the price and quantity of each product and showing the total sales.
